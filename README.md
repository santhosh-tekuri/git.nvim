status view
- 'j/k' or '<down>/<up>' select next/previous item
- 'g/GG' select first/last item
- 'o' or '<enter>' open diff view
- 'q' or '<esc>' close view

diff view
- sign column highligt indicates staged/unstaged
- '<tab>>' toggle staged/unstaged
- selection modes: hunk(default) or line
- 'v' toggle selection mode
- 'j/k' or '<down>/<up>' select next/previous item
- 'g/GG' select first/last item
- '<space>' stage/unstage selection
